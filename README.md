### 4️⃣ [Banking Customer Churn Prediction Pipeline](./Customer_Churn_Prediction_Pipeline)

* **Objective**: Predict customer churn and support proactive retention strategies through machine learning
* **Features**:

  * Analyzed 10,000 banking customer records to identify churn drivers
  * Performed exploratory data analysis uncovering patterns in geography, age, balance, activity level, and product usage
  * Engineered business-focused features including Balance-to-Salary Ratio, Products Per Year, Age Group segmentation, and High-Value Customer indicators
  * Built a reusable Scikit-learn preprocessing pipeline with scaling and one-hot encoding
  * Trained and evaluated Logistic Regression and Random Forest models
  * Optimized probability thresholds to achieve **90.6% precision** for churn identification
  * Developed an interactive Streamlit application for real-time churn prediction
  * Saved production-ready models using Joblib for deployment
* **Files**: `.ipynb`, `.csv`, `.pkl`, `.py`, `README.md`
